Stunning showcase of previously completed projects

The website jellymiso (alias of Jeremy Soh), is a portfolio platform for showcasing previously completed projects. Built with ReactJS, this website is made to be lightweight, fast, and reusable. Plans of expanding it further as a personal brand are underway, with the most recent changes being the depreciating of popup contents for React Router API.
